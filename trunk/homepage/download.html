Downloads

Binary

This is the first netPanzer release that runs under GNU/Linux, it should also be compilable under Win32 with Mingw, see below for informations on how to obtain the source code. The binary does not depend on non-standard libraries, so it should be useable on most GNU/Linux systems out of the box.

  • netPanzer Binary GNU/Linux - netpanzer-0.1.2-binary-linux-i386.tar.bz2 (11MB)

Debian

If you're using Debian unstable (Sid), you can add the following lines to your /etc/apt/sources.list file:

deb http://mentors.debian.net/debian unstable main contrib non-free
deb-src http://mentors.debian.net/debian unstable main contrib non-free
Second line is needed only if you intend to download sources of packages. After this modification you should run as root apt-get update and then apt-get install netpanzer.

Debian packages provided by Bartosz Fenski aka fEnIo fenio@o2.pl

Source Code

You need both the gamedata and the source if you want to compile yourself, in addition you need Jam, PhysicsFS, SDL, SDL_net, SDL_mixer and SDL_image, see the README for details.

  • netPanzer Source - netpanzer-0.1.2.tar.bz2 (350KB)
  • netPanzer Gamedata - netpanzerdata-0.1.2.tar.bz2 (9.32MB)

Older releases and other files are available via the Berlios's Filelist.
